##  À propos du projet

Ce dépôt est dédié à l’apprentissage approfondi du langage **Java** à travers des travaux pratiques, des exercices progressifs et des mini-implémentations orientées objet.

L’objectif principal est de construire une base solide en :
- Programmation Orientée Objet (POO)
- Architecture logicielle
- Bonnes pratiques de développement
- Résolution de problèmes
- Organisation de projets Java avec Maven

Ce repository représente également ma progression personnelle en tant qu’étudiant et développeur.

---

##  Objectifs

- Renforcer les bases du langage Java
- Maîtriser les concepts fondamentaux de la POO
- Comprendre les exceptions, lambdas et concepts avancés
- Écrire un code plus propre et structuré
- Développer une logique d’analyse solide
- Construire progressivement un niveau avancé en développement backend Java
